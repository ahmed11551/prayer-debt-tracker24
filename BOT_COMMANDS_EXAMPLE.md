# 📝 Примеры команд для Telegram бота

## Команды для BotFather (/setcommands)

Скопируйте и вставьте в BotFather:

```
start - 🚀 Начать работу с ботом
qaza - 📊 Пропущенные намазы (Каза)
dhikr - 📿 Зикры (Дуа, Азкары, Салаваты)
calculate - 🧮 Рассчитать долг намазов
progress - 📈 Мой прогресс восполнения
calendar - 📅 Календарь намазов
goals - 🎯 Мои цели и привычки
reports - 📄 Отчёты и статистика
help - ❓ Помощь и инструкции
settings - ⚙️ Настройки приложения
```
